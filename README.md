# Gradient-Distributing-Methods-for-Martian-Electron-Density-Prediction-Using-Big-Data-Frameworks
This project uses XGBoost, LightGBM, and CatBoost integrated with Apache Spark and Dask to predict Martian electron density measurements. The models were trained on data from the Mars Global Surveyor (MGS), MARSIS, and MAVEN missions.
